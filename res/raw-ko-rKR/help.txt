아케이드 센터

1.추가게임

  플레이어는 아케이드 게임 롬 직접 로드 선수 를 용이하게하기 위해 중심의 위치 를 감지 , ( 전화 실행중인 게임 센터는 SD 카드를 삽입해야합니다 ) 게임 롬 SD 카드 에 배치 할 수 있습니다.
         
2.치트 기능 :

아케이드 게임 속임수 속임수 센터 오픈 기능을 제공합니다. 방법 을 시작 :실행중인 게임 의 인터페이스에서 치트 치트 치트 치트 특정 인터페이스 옵션을 열 수 있는 옵션 메뉴에서 속임수 를 내 뿜었다 .
         
3.주요 기능 의 조합 :

기능 키 조합으로 동시에 여러 버튼 효과 를 누르면별도의 버튼 기능 으로 변환 될 수있다. 정품 인증 방법 은 : 게임 실행 인터페이스에서 키 조합을 설정 패널 , 변환 할 수있는 키 의 요구 에 하나의 버튼 의 효과를 확인하기 위하여 위로 하고 왼쪽 아래로 슬라이드 키 조합 레이아웃 버튼을 열고 키 조합 을 선택 , 옵션 메뉴를 내뿜어 , 레이아웃의 오른쪽은 여러 선택이 될 수 있습니다 키 조합 . 플레이어는 여러 개의 키 키 조합을 설정할 수 있습니다.

4.키 매크로 빨강 을 수행 할 수 있습니다
         
당신은 여러 번 사용하는 버튼 의 특정 순서와 별도의 버튼 을 통해 매크로 기능 키 플레이어 를 기록 할 수 있습니다. 정품 인증 방법 은 : 게임 인터페이스 를 실행 에서 옵션 메뉴를 내뿜어 , 매크로 버튼을 선택 , 매크로 설정 버튼 레이아웃을 열고 , 버튼 레이아웃 매크로는 움직 하고 변환 할 수있는 키 의 요구 에 하나의 버튼 의 효과를 확인하기 위해 왼쪽 버튼을 아래로 , 레이아웃 의 오른쪽 부분을 기록하는 키 조합 입니다 기록 오류가 하나의 버튼 해지 조치를 주문하려면 오른쪽 아래 버튼으로 취소 할 수 있지만 , 선수 , 기록하기 위해조이스틱 및 버튼을 아래로 작동 할 수 있습니다. 플레이어는매크로 버튼 키 , 키와 키 조합 을 동시에 활성화 할 수 없습니다 를 기록 할 수 있습니다.
         
5.게임 읽기 기능을 저장합니다 :

아케이드 게임 센터는 현재 즉시 보관 하고 읽을 수 있습니다. 정품 인증 방법 : 게임 실행 인터페이스에서 새로운 아카이브가 새로운 게임 아카이브를 저장합니다 생성 할 수있는 옵션 을 클릭 , 이미 게임 저장 기존의 범위를 보관됩니다 클릭 아카이브 (파일 읽기) , 아카이브 인터페이스를 선택 , 옵션 메뉴를 내 뿜었다 . 파일 인터페이스 옵션 을 읽는 당신은 이미 저장된 파일을 읽을 수있는 게임 파일을 선택할 수 있습니다.
        
6.블루투스 게임 :

플레이어 가 블루투스 를 켜야 할 다른 플레이어 에 의해 발견 , 선택은 연결 모드 로 다른 플레이어 를 클릭 연결이 게임 을 시작 하기 에 성공 , 대기 모드로 전환 하는 것입니다 (참고 : 사람이 게임 패키지 에 존재하지 않을 경우 , 파일 전송 모드로 전환 을 , 파일 전송 이 완료 바로 게임이 시작 ) .
        
7.네트워크 플레이 :

대기 모드로 전환 하려면 예를 선택 , 다른 플레이어 에 의해 발견 ,연결 모드 로 다른 플레이어 를 클릭 연결이 게임을 시작하는 성공 (주 : 사람이 게임 패키지 에 존재하지 않을 경우 , 파일 전송 모드로 전환 을 , 파일 전송 이 게임을 시작 직접 수행된다 )
